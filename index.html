Listas

Temos dois tipos de listas no HTML: ordenadas e não ordenadas.

Listas não-ordenadas - tag UL - Unordered List

  • list item
  • item da lista
  • outro item
  • mais um item...

Listas ordenadas - tag OL - Ordered List

  1. list item
  2. item da lista
  3. outro item
  4. mais um item...

Essas listas podem ter como itens quaisquer valores, não apenas texto.

Também podemos misturar valores nos itens da lista.

  • Texto, texto...
  • Texto com imagem:

Exemplo: Uma lista de links.

  1. link para alguma coisa
  2. link para alguma coisa
  3. link para alguma coisa
  4. link para alguma coisa
  5. link para alguma coisa
